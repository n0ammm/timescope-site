Politique de Confidentialité

Dernière mise à jour : 21 décembre 2025

En résumé : TimeScope respecte votre vie privée. Vos données de calendrier restent sur votre appareil et ne sont jamais envoyées à nos serveurs.

1. Données collectées

TimeScope collecte les données suivantes :

  • Événements de calendrier — Accès en lecture seule à vos événements Google Calendar pour calculer le temps passé. Ces données restent localement sur votre navigateur.
  • Adresse email — Utilisée uniquement pour l'achat de la licence Premium et l'activation de votre clé de licence.
  • Préférences — Calendrier sélectionné, objectifs hebdomadaires, thème (clair/sombre). Stockées localement dans votre navigateur.

2. Comment nous utilisons vos données

Vos données sont utilisées exclusivement pour :

  • Calculer et afficher vos statistiques de temps
  • Sauvegarder vos préférences entre les sessions
  • Traiter les achats de licence Premium

3. Stockage des données

Toutes les données de calendrier et préférences sont stockées localement dans votre navigateur via l'API Chrome Storage. Aucune donnée d'événement n'est envoyée à des serveurs externes.

Seules les informations suivantes transitent par nos serveurs :

  • Votre email (pour créer le lien de paiement Lemon Squeezy)
  • Votre clé de licence (pour valider l'activation Premium)

4. Partage des données

Nous ne vendons, n'échangeons ni ne transférons vos données personnelles à des tiers, sauf dans les cas suivants :

  • Lemon Squeezy — Notre processeur de paiement reçoit votre email et informations de paiement pour traiter les achats Premium. Leur politique de confidentialité est disponible sur lemonsqueezy.com/privacy.

5. Sécurité

Nous prenons la sécurité de vos données au sérieux :

  • Authentification via Google OAuth 2.0 (standard de l'industrie)
  • Communications chiffrées via HTTPS
  • Clés de licence stockées de manière sécurisée sur Upstash Redis
  • Aucun mot de passe stocké — nous utilisons uniquement les tokens Google

6. Vos droits

Vous pouvez à tout moment :

  • Supprimer vos données — Désinstallez l'extension pour supprimer toutes les données locales
  • Révoquer l'accès — Allez sur myaccount.google.com/permissions pour révoquer l'accès de TimeScope à votre calendrier
  • Demander la suppression — Contactez-nous pour supprimer votre clé de licence de nos serveurs

7. Modifications

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications seront publiées sur cette page avec une nouvelle date de mise à jour.

8. Contact

Pour toute question concernant cette politique de confidentialité :
